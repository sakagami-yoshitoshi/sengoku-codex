# Sengoku Codex

一个用于对照日本现代行政区划与战国时代旧地名的静态网页地图项目。

## 内容

- `outputs/japan-sengoku-map.html`
  - 主网页文件
- `outputs/japan-prefectures.svg`
  - 地图矢量底图

## 使用方式

直接在浏览器中打开 `outputs/japan-sengoku-map.html` 即可浏览。

## 特点

- 以现代日本都道府县为基础进行交互浏览
- 以日文/英文双语显示现代地名与战国地名
- 点击都道府县后可切换到局部放大视图
- 默认全域视图下显示日本八大区域配色

## 说明

本项目当前以桌面端浏览体验为主。
